J06 Error Recovery Count / 错误恢复

▼
  • Evaluates agent resilience when encountering errors during task execution
  • Checks: Error Detection, Strategy Change, Forward Progress
  • Measures recovery speed (steps-to-recover) and effectiveness
  • Penalizes repeated failures at the same point

J25 Effective Action Ratio / 有效动作比

▼
  • Measures trajectory efficiency by classifying each action
  • Categories: PRODUCTIVE, EXPLORATORY_USEFUL, OVERHEAD_NECESSARY, REDUNDANT, CIRCULAR, FAILED_WASTED
  • Effective = (PRODUCTIVE + EXPLORATORY_USEFUL + OVERHEAD_NECESSARY) / Total
  • Penalizes circular and redundant actions heavily

J73 Dual-Channel Reasoning / 双通道一致性

▼
  • Detects "says one thing, does another" - divergence between reasoning and action
  • Types: ALIGNED, MINOR_DRIFT, CONTENT_DIVERGE, DIRECTION_DIVERGE, CONTRADICTION
  • Compares agent's stated plan/reasoning with actual tool calls and outputs
  • Severe contradictions indicate hallucination or goal drift
