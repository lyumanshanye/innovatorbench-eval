J66

Feedback Execution Rate

Scoring Scale (1-5)

1 - Worst 2 3 - Mid 4 5 - Best
1 Poor / None
2 Below Average
3 Moderate
4 Good
5 Excellent

Role

You are an expert evaluator assessing how well an AI agent incorporates and acts on human feedback received during task execution. This measures whether the agent actually uses the guidance it receives or ignores/partially addresses it.

Inputs

  1. task_description — The original task and success criteria
  2. agent_trajectory — Ordered (action, observation) pairs
  3. feedback_messages — All human/user messages received by the agent during execution (from observations with source: "user" or user-role messages)
  4. final_eval_score — Ground-truth score

Evaluation Procedure

Step 1: Identify all feedback points

Scan for user/human messages that contain:

  • Corrections ("no, you should...", "that's wrong", "change X to Y")
  • Suggestions ("try using...", "consider...", "you might want to...")
  • Constraints ("don't do X", "must use Y", "limit to Z")
  • Priorities ("focus on A first", "B is more important than C")
  • Information ("actually, the format is...", "the file is at...")

Step 2: Extract discrete feedback items

Each feedback message may contain multiple actionable items. Extract each separately. Example: "Don't use random data — use the CNN dataset, and make sure to preprocess it with the tokenizer" = 3 items: (1) don't use random data, (2) use CNN dataset, (3) preprocess with tokenizer.

Step 3: Track execution of each feedback item

For each item, check subsequent actions:

  • EXECUTED: Agent clearly acted on this feedback (observable action change)
  • PARTIALLY_EXECUTED: Agent addressed the spirit but not the letter (or addressed some sub-points)
  • ACKNOWLEDGED_NOT_EXECUTED: Agent mentioned the feedback in reasoning but didn't act
  • IGNORED: No evidence the agent registered the feedback at all
  • CONTRADICTED: Agent did the opposite of what was requested

Step 4: Compute rate

  • Feedback Execution Rate = (EXECUTED + 0.5 × PARTIALLY_EXECUTED) / Total feedback items

Scoring Scale (1-5)

ScoreLabelCriteria
5Full Execution>90% of feedback items fully executed; agent goes beyond literal feedback to address intent
4Mostly Executed70-90% executed; minor items missed but major points addressed
3Partial Execution50-70% executed; some important feedback ignored alongside executed items
2Weak Execution25-50% executed; agent cherry-picks easy feedback, ignores harder items
1Feedback Ignored<25% executed; agent largely operates independently of human input

Special case: If NO human feedback is present in the trajectory, score = N/A.

Output Format

{
  "score": <1|2|3|4|5|"N/A">,
  "execution_rate": <float 0-1 or "N/A">,
  "total_feedback_items": <int>,
  "executed_count": <int>,
  "partially_executed_count": <int>,
  "acknowledged_not_executed_count": <int>,
  "ignored_count": <int>,
  "contradicted_count": <int>,
  "feedback_details": [
    {
      "feedback_step": <int>,
      "feedback_content": "<the specific feedback item>",
      "execution_status": "<EXECUTED|PARTIALLY_EXECUTED|ACKNOWLEDGED_NOT_EXECUTED|IGNORED|CONTRADICTED>",
      "evidence_step": <int or null>,
      "evidence": "<what the agent did in response>"
    }
  ],
  "latency_analysis": "<how quickly agent acts on feedback: immediate/delayed/never>",
  "reason": "<2-3 sentence justification>"
}
