How many tool-using steps the agent took. Each react node is one (think|action, observation) pair.

At a Glance

FieldValue
Metric IDM02
EnglishTool Chain Depth
中文工具链深度
MethodAuto / Rule
Value range≥ 0 (integer)
DirectionNeutral — extreme high suggests inefficiency, very low suggests under-exploration

Formula

M02 = len(react_nodes)

Required Inputs

node.node_type == 'react'

Procedure

  1. Iterate over the trajectory's nodes.
  2. Count nodes whose node_type == 'react'.
  3. Return the integer count.

Worked Example

Primary trajectory has 225 react nodes ⇒ M02 = 225.

Edge Cases

  • Trajectory with 0 react nodes (planning-only) ⇒ M02 = 0.
  • Cancelled / incomplete trajectories still count whatever react nodes exist.

Output Keys

  • M02_tool_chain_depth
