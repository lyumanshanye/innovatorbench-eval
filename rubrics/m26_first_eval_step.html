Eval at the end

When did the agent first invoke eval? Early evals reveal a tighter feedback loop; very late evals indicate the agent waited until the end.

At a Glance

FieldValue
Metric IDM26
EnglishFirst Eval Step
中文首次评估步骤
MethodAuto / Rule
Value range≥ 0 (integer step index)
DirectionLower = earlier feedback loop (good); but too early may waste evals

Formula

M26 = first index i in react_nodes where action.action_type == 'eval', else None

Required Inputs

action.action_type == 'eval'

Procedure

  1. Iterate react nodes in order.
  2. Return the index of the first 'eval' action, or None if none.

Worked Example

First 'eval' at react node[145] ⇒ M26 = 145 (≈ 64% through the 225-step trajectory).

Edge Cases

  • 0 eval actions ⇒ None.
  • Index is 0-based across react nodes only.

Output Keys

  • M26_first_eval_step
